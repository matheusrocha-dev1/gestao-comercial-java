# Sistema de Gestão Comercial

Fiz esse projeto pra praticar Java na prática mesmo, saindo só da teoria.

A ideia foi montar um sistema simples de cadastro, onde dá pra gerenciar clientes e produtos direto pelo console.

## O que o sistema faz

- Cadastrar clientes
- Listar clientes
- Atualizar dados dos clientes
- Remover clientes

- Cadastrar produtos
- Listar produtos
- Atualizar produtos
- Remover produtos

## Tecnologias

- Java
- ArrayList
- Scanner
- Um pouco de POO

## Por que fiz esse projeto?

Queria parar de só assistir aula e realmente começar a programar.  
Quero mostrar que estou evoluindo.

Enfrentei vários erros (principalmente em saber o que fazer e em algumas partes de como fazer), mas fui resolvendo um por um.

Esse projeto marca um ponto importante pra mim:

- Começar a construir coisas sozinho  
- Aprender novas funções do Java  
- Consolidar o que eu já sabia  
- E o principal... mostrar até pra mim mesmo que estou aprendendo bem. Só depende do meu esforço e vontade, e ver tudo que fiz me motiva muito a fazer mais e mais      

Agora nos próximos passos quero evoluir esse sistema aplicando mais conceitos de orientação a objetos, como herança e polimorfismo, além de organizar melhor o código (sei que dá pra melhorar muito kk)
